To read a CK3 save file :

  1. Save game manually, automatic save are different.
  2. Edit your save file with Notepad++ or other editor to remove the text portion. Saves are normally located at "Users\[your username]\Documents\Paradox Interactive\Crusader Kings III\save games\".
  3. Save what's left into a new file and add the .zip suffix.
  4. Unzip the new file and you should get a file name dataset.
  5. Upload that file with the button below.
