CK3 Visualizer

Import your Crusader Kings III save and view multiple fun stats!

CK3 Visualizer is fan made project. It's made using Angular, a javascript framework, and Plotly a free extensive javascript chart maker.

This project is open source and you are welcome to contribute via : Github Ck3 Visualizer

All users, event non technical can suggest charts or other ideas via this Trello board
